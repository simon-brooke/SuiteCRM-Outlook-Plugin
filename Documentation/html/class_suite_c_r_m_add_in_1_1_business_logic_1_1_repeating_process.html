virtual int SuiteCRMAddIn.BusinessLogic.RepeatingProcess.PrepareShutdown ( )
virtual

Put me into a mode where I finish all the work I have to do quickly.

This method will be called repeatedly; overrides should be written with this in mind.

Returns
Zero if I may be stopped immediately (this is the default); otherwise an integer indicating the number of work units to complete before I can be stopped.
