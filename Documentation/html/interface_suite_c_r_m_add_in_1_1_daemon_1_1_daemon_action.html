Detailed Description

An action to be queued and performed by the DaemonWorker.

DaemonActions are intended to be run essentially once, but may be allowed a number of attempts (intended to be limited) in case, for example due to network problems, the first attempt(s) fail. However, DaemonActions are not intended for things which are to be run repeatedly. For that, specialise RepeatingProcess.
