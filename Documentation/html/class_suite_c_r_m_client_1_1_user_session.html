Construct a new instance of a UserSession. Note that all these parameters (except log) come from the settings object, and it would be much simpler to just pass that in; unfortunately, that's in the SuiteCRMAddIn assembly, and that is dependent on this, so can't be included. TODO: see if this could be refactored.

Parameters
URLThe URL of the rest handler to connect to.
UsernameThe username to authenticate as.
PasswordThe password to authenticate with.
ldapKeyThe LDAP key to authenticate with.
logThe logger to log to.
timeoutThe timeout for calls to the URL.
