Detailed Description

A generic handler for a REST service. Derived in large part from Andrew's CrmRestServer, q.v., but that is static and I'm really unhappy about the number of static classes in this design. In the long term I'd like to make CrmRestServer a subclass of this class, but that is a major change I'm not yet ready to make.
