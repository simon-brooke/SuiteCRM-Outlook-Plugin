Detailed Description

A list which will not be corrupted if used by two or more threads concurrently.

C# does not provide a ConcurrentList class which shares the same API as IList; in particular ConcurrentBag (the nearest equivalent) has no Exists predicate. So I've rolled my own; this may not be maximally efficient but it should be safe.
